      gitctx #

      gitctx is a tool to manage git contexts

      Synopsis #

      gitctx helps you manage multiple git configurations and contexts easily. It allows you to switch between different git user profiles, repositories, and settings quickly.

      If this is your first time using gitctx, you can initialize it with the ‘gitctx init’ command. It will set up the necessary configuration files and directories and store your current git configuration.

      For more information, visit https://alexandrelithaud.github.io/gitctx/
